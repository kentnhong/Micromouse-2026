## MMS Simulation

This is the mock test simulation of the floodfill algorithm.

--- 

### How to build the simulation

To build this mock test simulation, navigate into the directory and run the compilation script:

**For Windows (PowerShell):**
```powershell
cd app/mms_sim
./sim.ps1
```

**For Linux/macOS (Bash):**
```bash
cd app/mms_sim
./sim.sh
```

*Note: This is only a mock test used for testing the floodfill algorithm.*
